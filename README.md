# Loaderjs
Loaderjs - javascript library created by Marrtan studio
http://marrtan.hys.cz

1) How to use
-	copy files to your location
-	insert your html into div with id "content"
-	now use App.Loader.loaded(loadedStatus) - loadedStatus must be true or false
